# supply-chain-late-delivery-risk
ML project to predict late delivery risk using decision-time information.

## Project Overview
This project predicts whether a customer order is at risk of late delivery using Machine Learning. The model uses only decision-time information to avoid data leakage and support early decision-making.

## Problem Statement
Late deliveries affect customer satisfaction and increase operational costs. This project helps identify orders that are likely to be delivered late before dispatch.

## Stakeholder
Supply Chain Manager / Logistics Manager

## Operational Decision
Identify orders that are at risk of late delivery before shipment so that preventive actions can be taken.

## Machine Learning Task
Classification

## Target Variable
Late Delivery Risk (Late / Not Late)

## Dataset Information
- **Dataset Name:** DataCoSupplyDataset.csv
- **Source:** Kaggle
- **Domain:** Supply Chain and Logistics
- **Type:** CSV
